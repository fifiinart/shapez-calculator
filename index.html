Shape Calculator


About

This tool will calculate the optimal way to create any shape in Shapez.io! Please keep in mind that for the shapes in the later levels of the game, there will be spoilers.

How to input a shape

Shapes can be generated with a short key. The short key of any shape in the game is copiable by clicking on the Copy Key button in the shape analysis screen like so:



Each shape consists of up to 4 layers, and each layers consists of 4 quadrants. Shape codes start from the upper right quadrant and move clockwise.


Each quadrant is described via a pair of letters - An uppercase letter describing the shape, followed by a lowercase letter describing its color. If the quadrant is empty, -- is used.


Layers are separated via :, starting with the lowest layer.

Shape codes

  • C Circle
  • R Rectangle
  • W Windmill / Fan
  • S Star

color codes

  • r Red
  • g Green
  • b Blue
  • y Yellow
  • p Purple
  • c Cyan
  • u Uncolored
  • w White
